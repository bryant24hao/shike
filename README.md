# 拾刻 ShiKe

> 拾取文学经典中的每一个时刻。

拾刻是一个 Chrome 新标签页插件——用中文经典文学中的时间描写显示当前时刻，以十六进制颜色为背景。

每次打开新标签页，你会看到一段与此刻对应的文学引文。可能是张继的"夜半钟声到客船"，可能是钱钟书写的"下午三点钟，夏天天长，离掌灯还早得多"，也可能是刘慈欣笔下某个精确到分钟的倒计时。背景色随时间流动——当前时间的十六进制值就是背景色（21:34:50 → `#213450`）。

---

## 功能

### 文学时钟
2500+ 条引文覆盖一天中的每一分钟。语料横跨古今中外：

- **中国古典**：《红楼梦》《诗经》唐诗宋词、四大名著、《史记》
- **中国现当代**：钱钟书、老舍、张爱玲、余华、莫言、路遥、王小波、刘慈欣
- **哲学思想**：博尔赫斯、加缪、木心、陈嘉映、尼采、海德格尔、奥勒留
- **科幻文学**：刘慈欣、阿西莫夫、特德·姜、菲利普·迪克、郝景芳
- **世界文学**：村上春树、东野圭吾、托尔斯泰、马尔克斯、卡夫卡、普鲁斯特
- **诗歌散文**：海子、北岛、顾城、余光中、三毛、龙应台、梁实秋

同一分钟有多条引文时随机展示，优先显示中文原创内容。

### HEX 颜色
时间即颜色。每一秒，背景色都在变化。支持原始模式（偏暗色调）和增强模式（更丰富的色域）。文字颜色遵循 WCAG 2.0 对比度算法自动适配，确保在任何背景色上都清晰可读。

### 图片背景
可选的图片纹理模式——21 张 CC0 风景照作为若隐若现的底纹，叠加在 HEX 颜色之上。根据当前时段自动切换风格：

| 时段 | 风格 |
|------|------|
| 日出（5:00-7:59） | 金色晨光、薄雾、朝霞 |
| 白天（8:00-16:59） | 山峰、云层、湖泊、森林 |
| 傍晚（17:00-19:59） | 橙紫色晚霞、海岸落日 |
| 夜晚（20:00-4:59） | 星空、暗色海面、银河 |

也支持上传自定义图片。

### 时间漫游
按 `←` `→` 键翻阅不同时刻的引文，像翻书一样浏览一天中的文学风景。按 `↑` `↓` 跳整点。按 `Esc` 回到现在。

### 虫洞穿越
按 `Space` 键启动虫洞——从当前时刻开始往回快速闪回，在 3 分钟内穿越一整天的文学引文。开头缓慢，中间加速，结尾减速停下，像被引力捕获。底部时间轴显示穿越进度。

### 导出为图片
点击右侧的"刻"字印章，将当前画面导出为 2x 高清 PNG 图片——所见即所得，包括背景色、纹理、霞鹜文楷字体。

## 安装

### 从 Chrome Web Store 安装
（审核中，即将上线）

### 从源码安装
1. 下载或 `git clone` 本仓库
2. 打开 Chrome，进入 `chrome://extensions`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择仓库中的 **`src/`** 目录

---

## 快捷键

| 按键 | 功能 |
|------|------|
| `←` `→` | 时间漫游：前后翻阅引文 |
| `↑` `↓` | 跳到上/下一个整点 |
| `Space` | 虫洞穿越：自动回溯 / 暂停 |
| `Esc` | 回到实时模式 |

鼠标悬停页面底部可以看到快捷键提示。

---

## 设置

右键插件图标 → 选项，可以调整：

- **颜色模式**：原始 / 增强
- **主题**：跟随系统 / 浅色 / 深色
- **字体**：文楷（LXGW WenKai）/ 楷体 / 黑体
- **背景模式**：纯色 / 图片（按时段自动切换，可自定义上传）
- **遮罩浓度**：调节图片纹理的可见程度

所有设置可一键恢复默认。

## 灵感来源

拾刻的创意融合了两个项目：

- [Literature Clock](https://literature-clock.jenevoldsen.com/) — 用英文文学段落显示时间的网站，由 The Guardian 读者众包 2300+ 条引文
- [Hex Clock](https://www.jacopocolo.com/hexclock/) — 用十六进制颜色值显示时间

拾刻将两者结合，做成了第一个**中文文学时钟**。
