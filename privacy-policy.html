拾刻 ShiKe — 隐私政策 ShiKe — Privacy Policy

最后更新：2026 年 3 月 18 日 Last Updated: March 18, 2026

概述 Overview

拾刻（ShiKe）是一个 Chrome 浏览器新标签页扩展。我们重视您的隐私，本扩展不收集、不存储、不传输任何用户个人数据。

ShiKe is a Chrome new tab extension. We value your privacy. This extension does not collect, store, or transmit any personal user data.

数据收集 Data Collection

本扩展不收集任何用户数据，包括但不限于：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史或搜索记录
  • 位置信息
  • 设备标识符
  • 用户行为分析数据

This extension does not collect any user data, including but not limited to:

  • Personal identity information (name, email, phone, etc.)
  • Browsing history or search history
  • Location data
  • Device identifiers
  • User behavior analytics

本地存储 Local Storage

本扩展使用 chrome.storage.local API 在您的设备本地存储以下偏好设置：

  • 颜色模式（原始/增强）
  • 主题偏好（跟随系统/浅色/深色）
  • 字体选择
  • 背景模式及自定义背景图片

这些数据仅存储在您的本地设备上，不会上传到任何服务器。

This extension uses the chrome.storage.local API to store the following preferences locally on your device:

  • Color mode (raw / enhanced)
  • Theme preference (system / light / dark)
  • Font selection
  • Background mode and custom background image

This data is stored only on your local device and is never uploaded to any server.

网络请求 Network Requests

本扩展不发起任何网络请求。所有资源（语料数据、字体文件、背景图片）均打包在扩展内部，完全离线可用。

This extension makes no network requests. All resources (quote data, font files, background images) are bundled within the extension and work fully offline.

第三方服务 Third-Party Services

本扩展不使用任何第三方分析、广告或追踪服务。

This extension does not use any third-party analytics, advertising, or tracking services.

权限说明 Permissions

本扩展仅请求 storage 权限，用于在本地保存您的偏好设置。不请求访问浏览历史、标签页内容或其他敏感权限。

This extension only requests the storage permission to save your preferences locally. It does not request access to browsing history, tab content, or any other sensitive permissions.

儿童隐私 Children's Privacy

本扩展不针对 13 岁以下儿童，也不故意收集儿童的个人信息。

This extension is not directed at children under 13 and does not knowingly collect personal information from children.

政策变更 Changes

如本隐私政策发生变更，我们将在此页面更新。重大变更会在扩展更新说明中通知。

If this privacy policy changes, we will update this page. Significant changes will be communicated through extension update notes.
